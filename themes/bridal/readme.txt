== Description ==

Bridal is a fully responsive WordPress theme specially cater for Transport, Bridal, Logistic company. Bridal has everything you need to build your website without having coding knowledge. Comes with one click demo install so that you don't required additionl imports. Our theme options give you almost unlimited freedom of color options. You can change everything on your theme. Included default slider in the theme with slider transition and animation options. Comes with easy to use customizer to customize every aspect of the theme. Compatible with latest version of WordPress, WooCommerce and other popular plugins.
